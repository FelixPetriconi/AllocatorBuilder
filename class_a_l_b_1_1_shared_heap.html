Detailed Description

template<class Allocator, size_t NumberOfChunks, size_t ChunkSize>
class ALB::SharedHeap< Allocator, NumberOfChunks, ChunkSize >

The SharedHeap implements a classic heap with a pre-allocated size of _numberOfChunks.value() * _chunkSize.value() It has a overhead of one bit per block and linear complexity for allocation and deallocation operations. It is thread safe, except the moment of instantiation. As far as possible only a shared lock + an atomic operation is used during the memory operations
