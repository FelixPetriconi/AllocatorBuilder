Detailed Description

template<class Allocator, unsigned Flags = ALB::StatsOptions::all>
class ALB::AllocatorWithStats< Allocator, Flags >

This Allocator serves as a facade in front of the specified allocator to collect statistics during runtime about all operations done on this instance. Be aware that collecting of caller informations can easily consume lot's of memory! With a good optimizing compiler only the code for the enabled statistic information gets created.

Template Parameters
AllocatorThe allocator that performs all allocations
FlagsSpecifies what kind of statistics get collected
