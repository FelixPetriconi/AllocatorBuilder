Detailed Description

template<size_t DefaultAlignment = 16>
class ALB::AlignedMallocator< DefaultAlignment >

This class implements a facade to the system ::malloc(). According the template parameter DefaultAlignment the allocated values are aligned to the specific boundary in bytes. Normally this should be a multiple of at least 4 bytes.

Template Parameters
DefaultAlignmentSpecified the alignment in bytes of all allocation and reallocations.
