Detailed Description

template<class Allocator, size_t NumberOfChunks, size_t ChunkSize>
class alb::heap< Allocator, NumberOfChunks, ChunkSize >

The Heap implements a classic heap with a pre-allocated size of _numberOfChunks.value() * _chunkSize.value() It has a overhead of one bit per block and linear complexity for allocation and deallocation operations.
